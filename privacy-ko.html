개인정보처리방침

Apnea Comp

최종 업데이트: 2026년 5월 3일

1. 소개

이 개인정보처리방침은 Apnea Comp 모바일 애플리케이션 ("우리", "당사", "본 앱")이 서비스 사용 시 정보를 어떻게 수집, 사용, 보호하는지 설명합니다. 본 앱은 AIDA 공인 프리다이빙 대회를 운영하는 주최자, 심판, 스태프를 위한 대회 관리 도구입니다.

본 앱을 사용하면 이 방침에 명시된 내용에 동의하는 것으로 간주됩니다.

2. 수집하는 정보

2.1 계정 정보

계정 생성 시 다음 정보를 수집합니다:

  • 이메일 주소 — 인증 및 계정 복구용
  • 표시 이름 — 같은 이벤트의 다른 참가자에게 표시됨
  • 비밀번호 — 인증 제공자(Supabase Auth)를 통해 안전하게 해시되어 저장됨. 평문으로 저장되지 않습니다.

2.2 프로필 정보 (선택)

  • 프로필 사진 — 업로드를 선택한 경우. 이 이미지는 가입한 이벤트의 다른 멤버에게 표시되며, 활동 로그에 참조됩니다 (섹션 2.5 참조).

2.3 선수 대회 데이터

AIDA International에서 이벤트를 가져올 때 본 앱은 다음을 저장합니다:

  • 선수 이름 (성, 이름)
  • 성별
  • 국적
  • 종목 (STA, DYN, CWT 등)
  • 출전 기록 (Announced Performance, AP)
  • 개인 최고 기록 (Personal Best, PB)
  • 대회 결과 (실현 기록, 카드, 페널티 사유, REMARKS)
  • 체크인 사인 — 선수가 스태프 기기에서 디지털로 입력한 서명

이 정보는 체크인 사인을 제외하고 AIDA International의 공개 대회 기록에서 공식 API를 통해 가져옵니다. 체크인 사인은 본 앱 내에서 직접 수집됩니다.

2.4 수집하지 않는 정보

다음 정보는 명시적으로 수집/저장하지 않습니다:

  • 선수의 이메일 주소
  • 선수의 전화번호
  • 선수의 거주지 주소
  • 선수의 생년월일
  • 선수의 사진 또는 얼굴 이미지
  • 선수의 AIDA UUID 또는 고유 식별자
  • 사용자 위치 정보
  • 추적용 디바이스 식별자
  • 검색 기록
  • 디바이스의 연락처

2.5 활동 로그

대회 운영의 투명성을 위해 본 앱은 이벤트 내에서 발생한 주요 작업을 기록합니다:

  • 로그 대상 — 판정 결과 입력, 일정 조정 (OT delay), 체크인 작업, 멤버 역할 변경
  • 각 로그 항목 내용 — 작업자의 표시 이름, 프로필 사진 URL (설정된 경우), 작업 종류, 영향받은 선수 이름 (해당 시), 시각
  • 공개 범위 — 같은 이벤트의 Organizer와 Main Judge에게만 표시됨
  • 보관 기간 — 이벤트와 연결되어 있으며, 이벤트 삭제 시 함께 삭제됨

활동 로그에는 위에 명시된 것 이외의 데이터가 포함되지 않습니다.

2.6 디바이스 로컬 저장 (오프라인 버퍼링)

풀/암반 환경처럼 인터넷이 자주 끊기는 곳에서도 운영을 이어갈 수 있도록, 본 앱은 판정 결과를 일시적으로 디바이스 메모리에 보관할 수 있습니다:

  • 저장 위치 — 앱이 실행 중인 동안 메모리에만 저장. 영구 저장소(스토리지)에 쓰지 않습니다.
  • 저장 시점 — 디바이스가 오프라인 상태에서 판정 저장이 시도될 때 자동으로
  • 저장 기간 — 서버와 동기화될 때까지. 인터넷 복구 후 수 초 내 자동 동기화됨
  • 공개 범위 — 동기화 성공 전까지 결과를 만든 디바이스에서만 보임. 같은 이벤트의 다른 디바이스에는 표시되지 않음
  • 제한사항 — 동기화 완료 전 앱 강제 종료 또는 디바이스 재부팅 시 오프라인 데이터는 손실됩니다. 본 앱은 해당 항목에 "⛔ Offline" 배지를 표시하고, 대기 중일 때 앱을 종료하지 말 것을 사용자에게 안내합니다.

이 버퍼링 동안 데이터는 제3자에게 전송되지 않습니다.

3. 정보 사용 방법

수집된 정보는 다음 용도로 사용됩니다:

  • 사용자 인증 및 계정 관리
  • 인가된 이벤트 스태프에게 대회 일정 및 선수 정보 표시
  • 판정 결과를 AIDA International에 제출 (이벤트 주최자가 설정한 경우)
  • 이벤트 스태프 디바이스 간 실시간 데이터 동기화
  • 이벤트 내 운영 투명성을 위한 활동 로그 유지
  • 본 앱 내 관련 이벤트 업데이트 알림

수집된 정보를 광고, 마케팅, 또는 제3자 판매 목적으로 사용하지 않습니다.

4. 데이터 공유

4.1 AIDA International과 공유

이벤트 주최자가 AIDA 연동을 API 토큰으로 설정한 경우:

  • AIDA International에서 이벤트 일자, 시작 명단, 선수 정보를 읽어옴
  • AIDA International로 판정 결과 (선수 기록, 카드, 페널티, REMARKS)를 제출

이 공유는 본 앱의 핵심 기능을 위해 필수이며, 이벤트 주최자에 의해 시작됩니다.

4.2 이벤트 내 공유

이벤트 참가자 (organizer, main judge, judge, staff)는 다음을 볼 수 있습니다:

  • 같은 이벤트의 다른 참가자 이름, 프로필 사진, 역할
  • 해당 이벤트의 선수 정보
  • 판정 결과의 실시간 업데이트
  • 활동 로그 (Organizer와 Main Judge만)

정보는 이벤트 단위로 격리됩니다 — 한 이벤트의 사용자는 본인이 속하지 않은 다른 이벤트의 데이터를 볼 수 없습니다.

4.3 제3자 서비스 제공자

본 앱 운영을 위해 다음 서비스를 사용합니다:

  • Supabase (supabase.com) — 백엔드 인증, 데이터베이스, 실시간 동기화, 프로필 사진 저장
  • AIDA International (aidainternational.org) — 공식 대회 데이터 출처
  • Apple App Store 및 Google Play Services — 앱 배포 및 충돌 보고

이 제공자들은 우리를 대신하여 데이터를 처리하며, 자체 개인정보처리방침의 적용을 받습니다.

4.4 법적 요구

법률, 법원 명령, 또는 사용자의 권리/재산/안전 보호를 위해 필요한 경우 정보를 공개할 수 있습니다.

5. 데이터 보관

  • 계정 데이터 — 계정이 존재하는 동안 보관. 언제든 삭제 요청 가능 (섹션 8 참조)
  • 이벤트 데이터 (선수, 결과, 로그, 사인) — 이벤트 주최자가 이벤트를 삭제하면 자동으로 함께 삭제
  • 오프라인 버퍼링 결과 — 디바이스 메모리에만 보관. 인터넷 복구 후 수 초 내 서버와 동기화되거나, 동기화 전 앱 종료 시 손실
  • 인증 토큰 — 단기간 유효. 자동 갱신 또는 만료

6. 데이터 보안

본 앱은 다음 여러 계층의 보안으로 정보를 보호합니다:

전송 및 저장

  • 모든 데이터는 암호화된 HTTPS 연결로 전송됩니다
  • 데이터는 인프라 제공자 (Supabase)에 의해 저장 시 암호화됩니다
  • 비밀번호는 업계 표준 알고리즘으로 해시됩니다. 평문 비밀번호는 절대 저장하지 않습니다

데이터베이스 수준 접근 제어

  • 사용자/이벤트 데이터를 포함한 모든 테이블에 Row-Level Security (RLS)가 적용됩니다
  • 사용자는 자신이 속한 이벤트의 데이터만 접근 가능
  • 승인 대기 중이거나 승인되지 않은 멤버는 이벤트 데이터에 접근 불가
  • 권한은 로그인 시점뿐 아니라 모든 쿼리마다 검증됩니다

역할별 권한

  • Main Judge / Organizer — 이벤트, 멤버, 대회 데이터 관리; 활동 로그 조회 가능
  • Judge — 대회 결과 조회 및 입력 가능
  • Staff — 이벤트 데이터 조회 가능
  • 권한은 이벤트별로 적용됩니다. 한 이벤트의 judge는 다른 이벤트에 접근 불가

AIDA API 토큰 보호

  • AIDA International과 동기화에 사용되는 AIDA API 토큰은 데이터베이스에 암호화되어 저장됩니다
  • main_judge와 organizer 역할만 토큰을 조회 가능. 제한된 데이터베이스 함수로 강제됩니다
  • 다른 역할 (judge, staff)은 이벤트의 다른 데이터에 접근 가능하더라도 토큰을 볼 수 없습니다
  • 토큰은 UI에서 마스킹되어 표시됩니다. 사용자가 명시적으로 표시를 선택해야 보입니다

세션 및 접근

  • 인증 토큰은 단기간 유효이며 자동 갱신됩니다
  • 사용자가 이벤트에서 제거되면 접근 권한이 실시간으로 회수됩니다
  • 계정 자격 증명은 사용자 본인의 책임입니다. 절대 공유하지 마세요

합리적인 보안 조치를 취하지만, 어떤 시스템도 완벽하게 안전할 수는 없습니다. 계정 자격 증명을 비밀로 유지하고 의심스러운 활동을 보고하는 것은 사용자의 책임입니다.

7. 아동 개인정보 보호

본 앱은 성인 이벤트 주최자, 심판, 스태프의 사용을 대상으로 합니다. 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. AIDA International의 대회 기록에는 다양한 연령의 선수가 포함될 수 있으나, 본 앱을 통해서는 누구의 (미성년자 포함) 연락처도 수집되지 않습니다.

부모 동의 없이 아동의 정보가 수집되었다고 의심되는 경우, 연락 주시면 삭제하겠습니다.

8. 사용자의 권리

사용자에게는 다음 권리가 있습니다:

  • 우리가 보유한 사용자 개인정보에 대한 접근
  • 본 앱의 프로필 설정을 통한 부정확한 정보 수정
  • 계정 및 관련 데이터 삭제
  • 본 앱 제거 및 계정 삭제 요청을 통한 동의 철회
  • 데이터 내보내기 (지원 요청 시)

이러한 권리를 행사하려면 섹션 11의 정보로 연락해 주세요.

9. 국제 데이터 전송

본 앱은 Supabase가 운영되는 지역 (현재 미국)에서 데이터를 처리하고 저장할 수 있습니다. 본 앱을 사용함으로써 사용자 데이터가 이러한 지역으로 전송되고 처리되는 것에 동의하게 됩니다.

10. 본 방침의 변경

이 개인정보처리방침은 수시로 업데이트될 수 있습니다. 중요한 변경이 있을 경우:

  • 상단의 "최종 업데이트" 날짜를 갱신합니다
  • 변경 사항이 중요하면 본 앱을 통해 사용자에게 알립니다

변경 후에도 본 앱을 계속 사용하면 업데이트된 방침에 동의하는 것으로 간주됩니다.
